## Current Strategy

```text
Greenroad = everyday goods + greener systems + custom goods.
Sustainability = north star.
Custom Goods = money-now lane, not the whole identity.
Thoughtful everyday goods = trust/content engine.
```

**Working phrase:** Custom merch and thoughtful everyday goods — grouped in one place.

**Work order:** docs first → existing sheet second → MCP third → site/code fourth.

The existing Greenroad Google Sheet is the control board. Do not create a separate custom goods sheet yet.

ZEYODA is internal-only parent/foundation memory and should not appear in public Greenroad copy unless Jai explicitly approves. GOSHBOT routes memory and warns about drift.
